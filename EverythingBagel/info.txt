EverythingBagel is a more centralized collection 
of random projects and ideas from class projects
and other directories in this repo.

matrixSolver takes an augmented matrix user input
and outputs the row reduced echelon form solution

minesweeper simulates a minesweeper game solely through
using user input

sudokuSolver is the Python port of my other Java
sudokuSolver

pokerSim is an attempt to make a simulated poker game 
with some additions, focusing on evaluating the ranks of
poker hands and comparing them to determine a winner
